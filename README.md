# vat-rates
Rest APIs to fetch standard and reduced vat rates
## How to run
1. Clone project 
```
git clone https://github.com/rupen-dhandhukiya/assessment.git
```

2. Run project
```
mvn spring-boot:run
```
3. Access swagger -
  http://localhost:8081/ce/swagger-ui/index.html
